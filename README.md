# How to commit

```bash
echo "# cpp-solving-hw-tasks" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:wasiful/cpp-solving-hw-tasks.git
git push -u origin main

```
